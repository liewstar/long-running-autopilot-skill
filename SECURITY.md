# Security Policy

## Supported versions

This project currently supports the latest `main` branch.

## Reporting a vulnerability

If you find a potential security issue, please do **not** open a public issue first.

Please report privately by email:

- `2935437378@qq.com`

Include:

- Affected file(s)/area(s).
- Reproduction steps.
- Impact and severity estimate.
- Suggested mitigation (if available).

We will acknowledge receipt as soon as possible and coordinate responsible disclosure.
