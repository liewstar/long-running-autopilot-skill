## [1.1.0] - 2026-02-10

### Added

- Bilingual repository docs with new `README.zh-CN.md`.
- Community health files: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `SECURITY.md`.
- GitHub collaboration templates for bugs, features, and pull requests.
- Local and CI skill validation via `tools/validate_skill.py` and workflow.

- Expanded `README.md` with installation, usage, validation, and contribution sections.
